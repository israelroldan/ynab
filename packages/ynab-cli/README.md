# ynab-cli 

A CLI tool to access YNAB data

## Installation

```sh
npm install ynab-cli --save
```


## Tests

```sh
npm install
npm test
```

## Dependencies

- [ynab](https://github.com/israelroldan/ynab): A Node.js library that provides native access to YNAB data

## Dev Dependencies
